What is OpenClaw?

A self-hosted AI gateway for coding agents

Key Features

  • Runs locally — on your machine or server
  • Multi-channel — Telegram, WhatsApp, Discord, iMessage
  • Agent-native — built for coding agents
  • Your data, your rules — no vendor lock-in

What You Get

  • Message your terminal from any chat app
  • Run git, shell, cron commands via chat
  • Schedule automated tasks
  • Persistent context across sessions

Get Started

Five minutes. Literally.

1. Install

brew install openclaw
# or
npm install -g openclaw

2. Connect a Channel

openclaw gateway
# Then connect Telegram/Discord/etc.

That's it. Your terminal now responds to messages.
