# OpenClaw Demo Repository

This is a demo repo used for the OpenClaw talk on April 22, 2026.

## What is this?

A simple repo to demonstrate OpenClaw's capabilities:
- Git integration from chat
- Automated commits and pushes
- Cron job triggers

## Try it yourself

```bash
# Clone this repo
git clone <repo-url>

# Make changes
echo "Hello from OpenClaw!" >> hello.txt
git add .
git commit -m "Add hello.txt"
git push
```
